# Asistente Digital Personalizado - Frontend
Este repositorio contiene el frontend de un Asistente Digital Personalizado (Nombre pendiente), diseñado para ofrecer una experiencia de usuario interactiva y profunda mediante el uso de tecnologías de procesamiento de lenguaje natural impulsadas por las API de OpenAI. Desarrollado con Angular 17, este proyecto se enfoca en proporcionar una interfazde configuracion y pruebas sencilla para el asistente.

## Tecnologías Utilizadas
[Angular 17](https://angular.io)
[Tailwind CSS](https://tailwindcss.com)
[Ricky0123 - VAD](https://github.com/ricky0123/vad)
[OpenAI Whisper 1](https://platform.openai.com/docs/guides/speech-to-text)
[OpenAI GPT-4 Turbo](https://platform.openai.com/docs/models/gpt-4-and-gpt-4-turbo)
[OpenAI Speech API - TTS](https://platform.openai.com/docs/guides/text-to-speech)
